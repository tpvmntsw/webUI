# USB Media Player 左右分欄介面更新

**日期**: 2026-06-23

## 變更摘要

將 USB Media Player 從原本的單一列表式介面改為左右分欄設計，右側採用 3x3 網格佈局。

## 變更內容

### 新增功能

1. **左右分欄佈局**
   - 左側面板 (280px)：顯示 Folder / Music / Video / Photo 四個類別
   - 右側面板：顯示選中類別的內容

2. **3x3 網格顯示**
   - 每頁顯示 9 個項目（3 欄 x 3 列）
   - 移除滾動條，改用分頁顯示
   - 底部顯示頁碼指示器（如 "1 / 2"）

3. **導航操作**
   - 左側：上/下鍵切換類別，右鍵/OK進入右側
   - 右側：上/下鍵以整行移動，左/右鍵水平移動
   - OK鍵：唯一進入資料夾的方式
   - 左鍵在第一欄：返回左側面板或上層資料夾

### 視覺優化

- 放大整體介面尺寸（寬度 92%、高度 85%）
- 標題字體 1.8rem
- 類別/檔案圖示 3.5rem
- 檔案名稱 1.2rem
- 選中項目使用藍色背景 (#3a86ff) 白色邊框

### 修正問題

- 修正選項超出畫面無法看到的問題（改用分頁）
- 修正在最右欄按右鍵會自動進入資料夾的問題（現在只有 OK 才會進入）

## 影響檔案

- `screens/app-usb.js`

## 測試步驟

1. Source 切到 USB
2. 輸入選擇 Scandisk 64KB
3. Insert USB 選 Scandisk 64KB 按 OK
4. 確認顯示左右分欄的 USB Media Player 畫面
5. 測試導航：上/下/左/右/OK/BACK
