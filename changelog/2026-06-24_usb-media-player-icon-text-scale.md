# USB Media Player 選中項目圖示文字放大效果

**日期**: 2026-06-24

## 變更摘要

為 USB Media Player 左右面板的選中項目添加圖示與文字放大動畫效果，提升視覺回饋。

## 變更內容

### 新增功能

1. **左側面板 (類別) 放大效果**
   - 選中時圖示放大 1.3 倍
   - 選中時文字放大 1.5 倍
   - 外框保持不變，僅內部元素放大
   - 使用 `.25s ease` 過渡動畫

2. **右側面板 (檔案/資料夾) 放大效果**
   - 選中時圖示放大 1.3 倍
   - 選中時文字放大 1.5 倍
   - 外框保持不變，僅內部元素放大
   - 使用 `.25s ease` 過渡動畫

### 設計原則

- 遵循 slides.html 設計規範
- 使用 CSS transition 實現平滑動畫
- 外框不變，避免遮蓋相鄰項目
- 游標移出時自動恢復原始大小

## 影響檔案

- `screens/app-usb.js`

## 測試步驟

1. Source 切到 USB，進入 USB Media Player
2. 在左側面板上下移動，確認選中類別的圖示和文字有放大效果
3. 進入右側面板，確認選中檔案/資料夾的圖示和文字有放大效果
4. 確認放大動畫平滑，移出時恢復正常大小
5. 確認放大後不會遮蓋相鄰項目
