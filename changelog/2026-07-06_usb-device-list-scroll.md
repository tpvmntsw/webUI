# USB Device List Scrolling

**Date:** 2026-07-06

## Summary

USB Device 列表新增滾動功能，超過 7 個項目時會顯示滾動條，游標移動時保持在視窗中間區域。

## Changes

### Scrolling Behavior
- 最多顯示 7 個項目
- 往下移動時，游標保持在倒數第二個位置
- 往上移動時，游標保持在第二個位置
- 超過 7 個項目時顯示滾動條

### Visual Styling
- 項目 padding: 22px 28px
- 項目 margin: 8px
- Icon: 2.4rem
- Name: 1.5rem
- Border-radius: 14px
