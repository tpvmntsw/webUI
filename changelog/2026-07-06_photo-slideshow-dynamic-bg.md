# 照片播放動態背景與簡化 HUD

**日期**: 2026-07-06

## 變更摘要

改進照片播放體驗：全螢幕背景改為靜態照片模擬，每張照片切換時背景跟著變化，並移除底部 HUD 中間的進度條。

## 變更內容

### 功能改進

1. **動態照片背景**
   - 新增 `getPhotoBackground(index)` 函數
   - 提供 8 種不同的 SVG 模擬照片背景：
     - 風景（山丘+太陽）
     - 海灘夕陽
     - 雪山
     - 星空夜景
     - 森林
     - 城市天際線
     - 沙漠仙人掌
     - 水下場景
   - 根據 `currentPlayingIndex` 循環使用不同背景

2. **簡化照片播放 HUD**
   - 移除底部 HUD 中間的「時間 + 進度條 + 總時長」
   - 僅保留：
     - 左側：播放狀態圖標 + 檔名
     - 右側：播放模式圖標 + 計數器 (1/N)

3. **移除大計時器顯示**
   - 照片播放時不顯示中央的大計時器
   - 影片/音樂播放仍保留計時器

4. **播放時間**
   - 照片預設 5 秒換下一張（維持原設定）

### 技術細節

- SVG data URI 中的單引號使用 `%27` 編碼避免 CSS 引號衝突
- HTML style 屬性使用單引號，CSS url() 使用雙引號

## 影響檔案

- `screens/app-usb.js`
  - 新增 `getPhotoBackground()` 函數
  - 修改 `renderPlayer()` 函數

## 測試步驟

1. Source 切到 USB，插入 USB 裝置
2. 進入 Photos 類別，選擇任一照片
3. 按 OK 開始播放
4. 確認背景顯示為靜態風景圖（非漸變）
5. 確認底部 HUD 只有檔名和計數器，無進度條
6. 等待 5 秒自動換下一張
7. 確認背景圖案跟著變化（8 種循環）
8. 測試影片播放，確認仍保留完整 HUD（含進度條）
