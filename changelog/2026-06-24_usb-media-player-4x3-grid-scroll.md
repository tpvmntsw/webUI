# USB Media Player 4x3 網格與平滑滾動更新

**日期**: 2026-06-24

## 變更摘要

將 USB Media Player 右側面板從 3x3 網格改為 4x3 網格，新增低調滾動條，實作平滑滾動邏輯讓游標盡量保持在中間列。

## 變更內容

### 功能修改

1. **網格佈局從 3x3 改為 4x3**
   - 每頁顯示 12 個項目（4 欄 x 3 列）
   - 移除原本的分頁指示器，改用滾動方式

2. **新增低調滾動條**
   - 右側顯示 6px 寬的半透明滾動條
   - 滾動條位置即時反映游標所在位置
   - 視覺低調，不影響主要內容

3. **平滑滾動邏輯**
   - 游標盡量保持在中間列（第二列）
   - 當游標移到第三列且下方還有內容時，自動向上滾動
   - 當游標移到第一列且上方還有內容時，自動向下滾動
   - 例如：共 20 項，游標從第二列往下時，畫面滾動顯示 5-16 項，游標在 9-12 位置

### Mock 資料擴充

1. **Folder 類別**：22 項（8 個資料夾 + 14 個檔案）
2. **Video 類別**：22 項（4 個資料夾 + 18 個影片）
3. **Music 類別**：22 項（4 個資料夾 + 18 個音樂）
4. **Photo 類別**：22 項（5 個資料夾 + 17 張照片）

### 視覺調整

- 檔案圖示：3.5rem → 3rem
- 檔案名稱：1.2rem → 1.1rem
- 項目內距：20px 12px → 16px 8px（配合 4 欄佈局）

## 影響檔案

- `screens/app-usb.js`

## 測試步驟

1. Source 切到 USB，進入 USB Media Player
2. 確認右側顯示 4x3 網格（12 個項目）
3. 確認右側有低調的滾動條
4. 按下鍵移動游標，確認：
   - 游標在第一列時往下，游標移到第二列
   - 游標在第二列時往下，畫面滾動，游標保持在第二列
   - 游標在第三列時（到底時），停止滾動
5. 按上鍵移動游標，確認反向滾動行為
6. 確認滾動條位置即時更新
